
# План: Элегантный эффект "блика" для карточек

## Проблема

Сейчас на карточках слишком много отдельных анимированных элементов:
- Мелькающие shimmer-линии сверху/снизу
- Пульсирующие углы
- Боковые свечения

Это создаёт визуальный шум. Нужен один красивый эффект - плавный "блик", как отражение света на стекле.

---

## Решение: Эффект "светового блика"

Создам анимированный градиент, который медленно "проходит" по карточке - как блик света на стеклянной поверхности. Переход от белого/прозрачного к бирюзовому и обратно.

---

## Что изменится

### 1. Новая анимация в tailwind.config.ts

Добавлю анимацию `glass-shine` - медленный (15 секунд) горизонтальный "блик":

```
"glass-shine": {
  "0%": {
    backgroundPosition: "-200% 0",
    opacity: "0"
  },
  "10%": {
    opacity: "1"
  },
  "50%": {
    backgroundPosition: "200% 0",
    opacity: "1"
  },
  "60%": {
    opacity: "0"
  },
  "100%": {
    backgroundPosition: "200% 0", 
    opacity: "0"
  }
}
```

### 2. Форма "Создать комнату" (Index.tsx)

**Убираю:**
- Пульсирующие углы (4 div с border)
- Пульсирующую границу

**Добавляю:**
- Один overlay с эффектом "блика" - диагональный градиент, который медленно перемещается
- Статичная тонкая граница с пульсацией opacity (не движущаяся)

### 3. Карточка профиля (ProfileCard.tsx)

**Убираю:**
- Shimmer-линии сверху и снизу
- Боковые glow-линии

**Добавляю:**
- Такой же "бликовый" overlay
- Чистый glassmorphism без мелькающих элементов

---

## Визуальный эффект

Карточка будет выглядеть как стекло с периодически "пробегающим" бликом:
- Блик появляется слева
- Медленно (15 сек) движется вправо
- Цвет: от белого/прозрачного через бирюзовый к прозрачному
- Угол: диагональный (45°)
- Пауза между бликами

Это похоже на эффект "Видеозвонки нового поколения" где текст переливается.

## Технические детали

### Tailwind config - новые keyframes:

```typescript
"glass-shine": {
  "0%, 100%": {
    transform: "translateX(-100%) rotate(-45deg)",
    opacity: "0"
  },
  "15%, 85%": {
    opacity: "0"
  },
  "40%": {
    opacity: "0.6"
  },
  "50%": {
    transform: "translateX(100%) rotate(-45deg)",
    opacity: "0.6"
  }
}

animation: {
  "glass-shine": "glass-shine 15s ease-in-out infinite"
}
```

### Overlay элемент:

```tsx
{/* Animated glass shine effect */}
<div 
  className="absolute inset-0 overflow-hidden pointer-events-none rounded-2xl"
>
  <div 
    className="absolute inset-0 animate-glass-shine"
    style={{
      background: 'linear-gradient(45deg, transparent 30%, rgba(255,255,255,0.1) 45%, rgba(6,182,212,0.15) 50%, rgba(255,255,255,0.1) 55%, transparent 70%)',
      width: '200%',
      height: '200%',
      top: '-50%',
      left: '-50%',
    }}
  />
</div>
```

## Результат

Обе карточки будут иметь:
- Чистый glassmorphism фон (bg-black/40 backdrop-blur-2xl)
- Статичная бирюзовая граница (border-primary/30)
- Один красивый эффект "блика" - медленный, ненавязчивый
- Никаких мелькающих углов или полос
